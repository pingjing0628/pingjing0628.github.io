補數

目的： 簡化減髮過程，縮減電路成本 作法： 利用 <b>“補數相加”“取代減法”</b> ex: A - B = A + B' => (-B) 定義： 一數值 N (整數 n 位，小數 m 位)，基底為 b

  1. b's 補數： (1) b^n - N, if N 不等於 0 (2) 0, if N = 0
  2. (b-1)'s 補數： b^n - b ^ -m - N

ex: (49.2)10, 求 10's 補數及 9's 補數 Sol: N = 49.2, b = 10, 整數位數 n = 2, 小數位數 m = 1 10's 補數： 10^2 - 49.2 = 50.8 9's 補數： 10^2 - 10^-1 - 49.2 = 100 - 0.1 - 49.2 = 50.7

快速解法： 欲求 b 進制的： (b-1)'s 補數：拿(b-1) 去減每一位數 b's 補數：將(b-1)'s 補數之最後位元 +1 即(b-1)'s complement + b^-m

ex: (1783.25)10, 求 10's 及 9's complement. Sol:

  1. 9's complement 正：10000 - 0.01 - 1783.25 = 8216.74 快：9999.99 - 1783.25 = 8216.74
  2. 10's complement 正：10000 - 1783.25 = 8216.75 快：8216.74 + 0.01 = 8216.75

ex1: 更快解 (10101000)2 求 1's 及 2's 1's: 01010111 => 1 -> 0, 0 -> 1 2's: 將最右 "1" bit 之前皆保留，其餘 1 -> 0, 0 -> 1 1010|1000 => 01011000

ex2: (0.01010)2 1's: 0.10101 2's: 0.010|10 => 0.10110
